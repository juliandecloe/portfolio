Discord Remake - Frontend Development

This schoolproject is a very free project. The name of the subject says it already: Frontend Development. We were allowed to choose a website ourselves which we then had to rebuild. We did have to choose between making the website fully responsive (usable for every screensize) or work on the surface plane (making the website more attractive and interactive). I chose for the surface plane, because even though I can make a website fully responsive, I had more fun working on the surface plane because then I could use more javascript, which I really enjoy doing.

I chose the Discord website, because it is a really impressive website with a lot of features.

The screen on the left side of this page is a prototype and can actually be used. Try to click a button! Don't know where to go? Just click next to the button and it shows you how to navigate!
