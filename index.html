Solar Discovery

Isn't space awesome? Well I think it is. That is why, during a school project, I made a multiplayer game where you can fly in space with your own rocket and discover planets. Using web sockets I made it so that multiple users can join and see eachother fly. The solar system in this game is a real scale model of our solar system using an API that gives data about every planet and all of their moons.

About me

Who am I?

My name is Julian de Cloe. I live in Haarlem, a city located in the Netherlands. I am 20 years old and in my last year of college. Right now I'm studying Communication and Multimedia Design at the Hogeschool van Amsterdam (HvA). This is a full time bachelor which lasts 4 years. During this study I found out that my passion lays with web development. This is a smaller branch within the bachelor. Even though I chose web development, I still had 2 years of college studying UX design, so you can say that I am also capable of UX research and design.

What I like about web development

I started back in 2019 with HTML and CSS. I quickly started to get the hang of it which made it immediately more fun to do. In the beginning of 2020 I learned how to use javascript (note that this was all front end development). My teacher was amazed about the fact that I learned it so quickly that the next year he offered me a job to tutor freshmen in javascript. I did this for three classes. I also did some sideprojects for fun and started coding in school projects where you didn't even have to code.

A year later I chose the minor Web Development. This is where I also started programming back end using Javascript. It is still the basics, because there is still so much more to learn an accomplish. The thing I like about back end is that it keeps you busy. There are a lot more brain teasers in the back end then in front end. But if I have to chose between front and back end, I would chose front end. Front end is more creative than back end. I still have a lot of creativity left in me that I want to express.

Doesn't it bring more joy to the user when a website is more entertaining to visit?
